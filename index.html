📢 제안 개요

본 제안서는 TimelyGPT 플랫폼의 에이전트 기능과 클라우드 인프라를 결합하여, 비전공자부터 IT 전공자까지 아우르는 단계별 AI 엔지니어링 교육 과정입니다. 총 3단계(기초-심화-확장)로 구성되어 있으며, 각 단계는 학생들의 디지털 리터러시 수준에 맞춰 설계되었습니다.

총 16주 과정 전 계열 맞춤형 클라우드 실습 포함

📍 통합 로드맵

Step 1

공통 기초 과정 (Platform Mastery)

4주 / 12시간

TimelyGPT 사용법 습득 및 프롬프트 엔지니어링 기초 확립

  • 플랫폼 온보딩 및 AI 윤리 교육
  • Role-Context-Task 구조화 프롬프트 작성법
  • RAG(문서기반 답변) 기초 및 미니 에이전트 제작
Step 2

전공 심화 과정 (Major-Specific)

8주 / 24시간

6대 계열별 전공 지식을 주입한 커스텀 에이전트 구축

  • 전공 데이터(논문, 판례 등) 수집 및 지식 베이스 구축
  • 심화 튜닝 (Chain-of-Thought 등) 및 페르소나 설정
  • 계열별 캡스톤 프로젝트 (환자 시뮬레이터, 법률 자문 봇 등)
Step 3

확장형 과정 (Cloud-Native Ops)

4주 / 12시간

클라우드 인프라 위에서 TimelyGPT API 연동 및 서비스 배포

  • AWS/Naver Cloud 인프라 구축 및 GPU 인스턴스 활용
  • TimelyGPT API 연동 및 외부 Vector DB 구축
  • 웹 서비스(Streamlit) 배포 및 최종 시연

Phase 1

공통 기초 과정 (AI Agent Platform Mastery)

전공 무관, 모든 학생이 갖춰야 할 AI 리터러시와 TimelyGPT 기초 활용 능력 배양

📅 상세 커리큘럼 (4주 / 12시간)

주차 주제 세부 교육 내용
1주차 플랫폼 온보딩 & AI 윤리
  • TimelyGPT 계정 생성 및 인터페이스(UI) 학습
  • 기본 챗봇 생성 및 대화 실습
  • AI 윤리: 할루시네이션(Hallucination) 인지 및 데이터 보안
2주차 프롬프트 엔지니어링 기초
  • AI 사고법: Role(역할), Context(맥락), Task(지시) 구조화
  • Zero-shot vs Few-shot 프롬프트 실습
  • 나만의 프롬프트 템플릿 만들기
3주차 RAG (검색 증강 생성) 기초
  • 지식 베이스(Knowledge Base) 개념 이해
  • 수업 교재(PDF) 업로드 및 문서 기반 질의응답 실습
  • AI가 내 문서를 잘 이해하도록 데이터 전처리하기
4주차 미니 프로젝트: My First Agent
  • 대학 생활 보조 에이전트 기획 (예: 시간표 관리, 과제 도우미)
  • TimelyGPT로 에이전트 배포 및 공유
  • 동료 에이전트 상호 평가 (Peer Review)
Phase 2

전공 심화 과정 (Major-Specific Agent)

내 전공 지식을 학습한 전문 에이전트 구축 및 워크플로우 자동화

📅 공통 커리큘럼 (8주 / 24시간)

주차 주제 세부 교육 내용
1-2주차 도메인 지식 구조화
  • 전공별 핵심 데이터(논문, 판례, 실험 데이터) 수집
  • TimelyGPT Knowledge Base 최적화를 위한 데이터 가공
  • 에이전트 페르소나(Persona) 상세 설정
3-4주차 심화 프롬프트 & 튜닝
  • Chain-of-Thought (생각의 사슬) 기법 적용
  • 전공 특화 프롬프트 테크닉 (예: 진단 로직, 코딩 컨벤션)
  • System Prompt 튜닝을 통한 답변 품질 개선
5-6주차 워크플로우 자동화
  • 멀티 에이전트 설계 (검색 + 작문 + 평가)
  • 에이전트 간 업무 인계(Handoff) 로직 구현
  • 복합 과제(Complex Task) 자동화 실습
7-8주차 캡스톤 프로젝트
  • 팀별 전공 해결사 에이전트 최종 개발
  • 실제 전공 과제(과제, 논문 등)에 투입하여 성능 검증
  • 최종 발표 및 데모 데이

🎓 계열별 에이전트 래핑 모델 (6대 계열)

⚙️

공학 계열

Tech Agent

"신뢰할 수 있는 기술 스펙만 검색하라"

  • Wrapping: 기술 DB 및 학술 사이트로 검색 소스 제한
  • Task: 최신 기술(6G, 로봇) 스펙 비교표 자동 생성
  • Outcome: 기술 트렌드 분석 리포트
🧪

자연과학 계열

Lab Agent

"우리 랩의 실험 프로토콜을 준수하라"

  • Wrapping: 실험 변수 의미 및 이상치 기준 학습
  • Task: CSV 데이터 업로드 시 자동 그래프 시각화
  • Outcome: 실험 결과 논문 초안 작성
⚕️

의학·보건 계열

Patient Persona

"까다로운 환자가 되어 나를 시험하라"

  • Wrapping: 특정 질환/성격을 가진 가상 환자 페르소나 주입
  • Task: 실시간 문진 및 상담 롤플레잉
  • Outcome: 임상 커뮤니케이션 스킬 강화
⚖️

법학 계열

Legal Advisor

"판례에 근거하여 논리를 펼쳐라"

  • Wrapping: 법령 및 판례 PDF 데이터베이스(RAG) 구축
  • Task: 계약서 독소 조항 검토 및 법리적 근거 제시
  • Outcome: 내용증명 및 소장 초안 작성
👥

사회과학 계열

News Clipper

"대중의 반응과 여론을 읽어내라"

  • Wrapping: 뉴스 댓글 및 소셜 미디어로 수집 범위 타겟팅
  • Task: 이슈별 긍/부정 여론 분석 및 요약
  • Outcome: 소셜 트렌드 인사이트 리포트
🎨

예체능 계열

Style Director

"나만의 예술적 스타일을 확장하라"

  • Wrapping: 선호하는 작가/화풍/색감 스타일 학습
  • Task: 추상적 아이디어를 구체적 프롬프트로 변환
  • Outcome: 창작 아이디어 무드보드 및 시안
Phase 3

확장형 과정 (Cloud-Native LLM Ops)

IT 전공자 대상, 클라우드 인프라와 TimelyGPT API를 연동한 서비스 구축

📅 상세 커리큘럼 (4주 / 12시간)

주차 주제 세부 교육 내용
1주차 Cloud Infra 구축
  • AWS / Naver Cloud 계정 설정 및 환경 구성
  • GPU 인스턴스 생성 및 기본 리눅스 세팅
  • 클라우드 보안 및 네트워크 기초
2주차 API 연동 및 확장
  • TimelyGPT API Key 발급 및 연동 실습
  • 외부 Vector DB (Pinecone, Milvus) 클라우드 배포
  • 대규모 문서 처리 파이프라인 구축
3주차 sLLM (소형언어모델) 실습
  • 오픈소스 sLLM (Llama 3, Gemma 등) 클라우드 로드
  • TimelyGPT와 sLLM의 하이브리드 활용 방안 연구
  • Fine-tuning 기초 맛보기
4주차 배포 및 서비스화
  • Streamlit 등을 활용한 웹 서비스 UI 구축
  • 클라우드 배포 및 API 서빙 (Serving)
  • 최종 프로젝트 시연
